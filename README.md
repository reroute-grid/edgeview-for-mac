## Key Features

### Wide Image Format Support

EdgeView supports many image formats, including:

* JPG, PNG, GIF
* TIFF, BMP
* PSD (Photoshop files)
* WebP, AVIF
* RAW camera formats
* PDF and illustration files

### Archive File Viewing

It can open images directly from compressed files without extracting them first, supporting:

* ZIP
* RAR
* 7Z
* CBZ
* CBR

### Comic and Manga Mode

Optimized reading features for comics:

* Page-by-page or continuous view
* Two-page spread mode
* Right-to-left reading support
* Smooth navigation and zoom

### Animation Support

Supports animated formats such as:

* GIF
* Animated PNG
* Animated WebP

### File Browser Integration

Allows browsing, organizing, renaming, and managing image files directly inside the app.

### Metadata Viewing

Displays image details such as camera settings, resolution, and file information.

### Fast Performance

Optimized for quick loading and smooth navigation, even with large image collections.

### Gesture and Shortcut Support

Supports trackpad gestures, keyboard shortcuts, and mouse controls for easy navigation.
